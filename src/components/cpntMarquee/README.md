### 一、用法
1. 在page.json中设置
    ```json
    {
      "usingComponents": {
        "cpnt-slider": "../../components/cpntMarquee/cpntMarquee"
      }
    }
    ```

2. 在wxml中使用
    ```html
    <cpnt-marquee item-height="50" height="150" dataset="{{marqueeList}}"/>
    ```

3. 在js中设置组件的值
    ```javascript
    Page({
      data: {
        marqueeList: [
          'Amy领取了 7元 红包',
          'Tom领取了 7元 红包',
          'Jenny领取了 7元 红包'
        ]
      }
    })
    ```

### 二、属性说明
| 属性名       | 类型    | 默认值       |描述                       |有效值      |
|:----------- |:------- |:-----------:|:------------------------- |:--------- |
| dataset     | Array   | ['暂无数据'] | 组件显示的文字内容          |           |
| height      | Number  | 50          | 组件高度                   |           |
| item-height | Number  | 50          | 一条文字的高度              |           |
